Identifie les segments avec le plus fort potentiel viral en t'appuyant sur les mecanismes dopaminergiques de l'attention. Un segment viral n'est PAS un segment "interessant" ou "bien dit" : c'est un segment qui declenche une erreur de prediction dans le cerveau du spectateur (Schultz, RPE) et ouvre un gap d'information qu'il a besoin de refermer (Loewenstein).

PRINCIPE FONDAMENTAL — la dopamine code l'imprevisibilite, pas le plaisir
Une phrase previsible = zero pic dopaminergique = scroll. Une phrase qui CASSE une prediction (contre-intuition, retournement, chiffre surprenant, aveu inattendu) = pic dopamine = retention. Tu cherches en priorite les moments ou le locuteur dit quelque chose que le spectateur ne s'attendait PAS a entendre.

CRITERES POSITIFS (privilegier)
1. Hook decisif dans les 3 premieres secondes du segment : une phrase d'ouverture qui annonce une promesse concrete OU une affirmation qui surprend. Formats qui marchent :
   - "Je croyais X, en realite Y"
   - "Voila comment je fais X sans Y"
   - "Tout le monde dit X, mais X est faux"
   - Chiffre choc + contexte ("J'ai perdu 40k en 3 mois parce que...")
   - Question rhetorique a forte tension ("Tu sais pourquoi 90% des gens echouent a X ?")
2. Gap d'information dose : le segment montre l'EFFET ou la CAUSE, jamais les deux. Si le locuteur dit "j'ai fait X et ca a marche grace a Y", le gap est ferme = pas de tension. Si il dit "j'ai fait X et ca a tout change" sans expliquer comment dans le segment, le gap reste ouvert = clic.
3. Tension non resolue : revelation amorcee mais pas completement deballe, anecdote dont la chute pique, contradiction explicite avec le sens commun, aveu de vulnerabilite ou d'echec concret.
4. Densite emotionnelle : colere, indignation, surprise, gene, fierte assumee, declaration tranchee. Le neutre est l'ennemi du viral.
5. Concret > vague : preferer "j'ai depense 47€ chez Lidl" a "j'ai fait des economies". Les nombres specifiques, les noms propres, les exemples precis battent toujours l'abstrait.
6. Punchline auto-portee : au moins une phrase du segment doit pouvoir etre extraite et fonctionner seule (citation, sous-titre, miniature).
7. MicroHooks internes : relances toutes les 3-5 secondes ("et la c'est le truc", "mais attends", "le pire c'est que"). Un segment plat apres le hook chute en retention vers 8s.
8. Architecture cognitive lisible : hook → promesse → preuve (mini-recit/exemple/chiffre) → chute. Un segment qui suit cette structure performe meme avec un contenu moyen ; un segment brillant mais sans architecture echoue.

CRITERES DE DISQUALIFICATION (rejeter meme si le contenu est bon)
- Phrase d'ouverture descriptive ou contextuelle ("alors aujourd'hui je vais vous parler de...", "donc en fait...", "comme je disais")
- Promesse vague non livree dans le segment ("ca a tout change ma vie" sans aucun concret)
- Segment qui DEPEND d'un contexte exterieur pour etre compris
- Conclusion morale generique ("il faut croire en soi", "tout est possible")
- Plus de 2 hesitations/repetitions/tics dans les 5 premieres secondes
- Pronom "nous" generique dominant (Upworthy : "nous" sous-performe vs "je"/"il"/"elle" qui implique le spectateur)
- Affirmation deja vue 1000 fois sur les reseaux (cliche)

HIERARCHIE DE SCORING
Privilegier dans cet ordre :
1. Segments avec contre-intuition forte + concret chiffre/specifique + chute claire (priorite max)
2. Segments avec aveu personnel inattendu + tension emotionnelle
3. Segments avec punchline auto-portee + structure hook/preuve/chute
4. Segments avec anecdote captivante meme sans contre-intuition

Si plusieurs segments couvrent le meme insight, garder UNIQUEMENT la version la plus dense (le hook le plus net, la formulation la plus tranchee). Pas de doublons d'idee.

REGLE ETHIQUE
Un segment viral doit livrer ce qu'il promet dans son hook. Un hook qui ouvre un gap jamais ferme par la suite = clic mais retention nulle = penalise par l'algo. Si le segment promet "voila comment j'ai fait X" mais ne donne aucun element concret, le rejeter meme s'il accroche.
